# Render Free

Repository co `render.yaml` de tao mot Web Service chay web, API va WebSocket cung mot domain, kem mot PostgreSQL Free tai Singapore.

1. Dang nhap Render bang GitHub.
2. Mo `https://dashboard.render.com/blueprints` va chon **New Blueprint Instance**.
3. Chon repository `nguyekie/gamenew`.
4. Xac nhan tao cac tai nguyen Free trong Blueprint.
5. Cho deploy hoan tat, sau do mo URL `https://aetherion-game.onrender.com` ma Render cap.

Gioi han Free: web service ngu sau 15 phut khong co traffic va co cold start; PostgreSQL Free het han sau 30 ngay; filesystem cua service khong ben vung. Chi dung cho demo/hobby, khong dung production.
