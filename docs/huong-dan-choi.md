# Hướng dẫn chơi Aetherion Strategy

## Mục tiêu

Phá Nhà chính của đối thủ hoặc khiến đối thủ đầu hàng. Trận bắt đầu sau 5 giây và sẽ tạm dừng tối đa 30 giây nếu một người mất kết nối.

## Bắt đầu

- Chơi với máy: mở **Trung tâm → Thi đấu → Luyện tập với máy**.
- Chơi PvP: hai người mở game, nhập cùng một mã phòng và nhấn **Vào phòng**.
- Xếp hạng: tạo hồ sơ trong mục **Thi đấu**, sau đó chọn tìm trận 1v1.

## Điều khiển

| Phím/thao tác | Chức năng |
| --- | --- |
| WASD | Di chuyển camera |
| Con lăn / chuột giữa | Thu phóng / kéo bản đồ |
| Chuột trái | Chọn quân hoặc kéo khung chọn |
| Shift + chuột trái | Chọn thêm quân |
| Chuột phải | Di chuyển hoặc tấn công mục tiêu |
| Ctrl + 1–5 / 1–5 | Lưu / gọi nhóm quân |
| Space | Đặt lại camera |
| H / X / R | Giữ vị trí / dừng / rút lui |
| F / Q / E / C | Đánh thường / Xung lực / bắn xuyên phá / pháo sáng |
| Z / O | Đổi đội hình / đầu hàng |
| F1–F5 | Chọn công trình để xây |
| J/K/L/N/M/P | Huấn luyện các loại quân |

## Mẹo chiến thuật

- Rừng làm quân khó bị phát hiện; đồi tăng tầm nhìn.
- Dùng pháo sáng trước khi đưa quân vào vùng chưa quan sát.
- Lưu các nhóm quân bằng Ctrl + 1–5 để điều khiển nhiều cánh quân nhanh hơn.
- Đặt điểm tập kết bằng cách nhấp chuột phải lên công trình.
